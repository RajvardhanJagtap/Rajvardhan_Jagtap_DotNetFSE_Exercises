# NUnitMoqHandsOn

This project contains two hands-on exercises for Digital Nurture .NET FSE:

- NUnit fundamentals with a Calculator class
- Moq-based unit testing with a CustomerService class

## Run the project

```bash
dotnet restore
dotnet build
dotnet test
```

## Notes

- The project targets .NET 10 so the test host matches the runtime installed on this machine.
- All tests are in the same test project for a compact hands-on setup.
